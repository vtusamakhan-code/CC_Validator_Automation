üìù Step-by-Step Process

1

Upload Main Folder

User selects the main folder containing all customer subfolders. System scans and identifies all image files.

2

Categorize Images (API Call)

System sends all images to the categorization API which identifies which images are credit cards and marks them as potential front or back.

3

Group Credit Cards (API Call)

API analyzes the categorized images and pairs front images with their matching back images to form complete card sets.

4

Merge Paired Images

For cards with both front and back, images are combined vertically (front on top, back below) for API processing.

5

Get Redaction Coordinates (API Call)

The merged image is sent to the OCR API which returns exact coordinates (positions) of card numbers and CVC codes.

6

Split Coordinates

System determines which coordinates belong to front image and which to back image based on vertical position.

7

Apply Redaction Separately

Black masks are applied to the ORIGINAL separate images (not merged) at the identified locations.

8

Save & Download

Both original and redacted versions are available for preview and download as separate files.
